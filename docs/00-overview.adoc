= Overview

Bebop FFI exposes Bebop binary serialization via a stable C ABI.

The key idea is to *avoid re-implementing the wire format in every language*.
Instead, any application calls into a small native library (Zig as canonical;
Rust welcome) that uses the canonical Bebop tooling for correctness.

One C ABI, multiple implementations, language-agnostic consumers.
